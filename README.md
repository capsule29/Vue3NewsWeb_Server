# Vue3NewsWeb_Server

-   运行方式

    -   pm2 运行

    ```sh
    npm run pm2
    ```

    > [pm2 文档](https://pm2.io/)
    > pm2 & bun -> run ts
    >
    > 你也可以执行用其他语言编写的 app ( fork 模式):
    > pm2 start app.ts -x --interpreter ts-node

    -   ts-node 运行

    ```sh
    npm start
    ```

# mysql 文件

`corporate_news_release_system.sql`，导入到 mysql 中即可使用。
